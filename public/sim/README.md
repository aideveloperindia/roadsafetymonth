# Simulation Assets

Place your PNG assets here. The following structure is expected:

## Required Assets

### Environments
- `env_road_2lane.png` - 2-lane road background
- `env_road_4lane.png` - 4-lane road background
- `env_intersection.png` - Road intersection
- `env_zebra_crossing.png` - Zebra crossing overlay
- `env_sidewalk.png` - Sidewalk/footpath
- `env_parking_bay.png` - Parking bay
- `env_school_zone.png` - School zone background

### Vehicles
- `veh_bike.png` - Single bike
- `veh_bike_with_3.png` - Bike with 3 riders
- `veh_bike_neutral.png` - Neutral bike
- `veh_car.png` - Car
- `veh_ambulance.png` - Ambulance
- `veh_fire_truck.png` - Fire truck

### Pedestrians
- `ped_male.png` - Male pedestrian
- `ped_female.png` - Female pedestrian
- `ped_child.png` - Child pedestrian
- `ped_phone_pose.png` - Pedestrian with phone
- `ped_neutral.png` - Neutral pedestrian

### Props
- `prop_signal_red.png` - Red traffic signal
- `prop_signal_green.png` - Green traffic signal
- `prop_signal_ped_walk.png` - Pedestrian walk signal
- `prop_signal_ped_stop.png` - Pedestrian stop signal
- `prop_sign_speed_30.png` - Speed limit 30 sign
- `prop_stop_line.png` - Stop line
- `prop_helmet.png` - Helmet
- `prop_bottle.png` - Bottle
- `prop_phone.png` - Phone
- `prop_trash.png` - Trash bin
- `prop_police_check.png` - Police check point

### UI Elements
- `ui_panel.png` - UI panel (for speed knob, etc.)

## Note

For now, the simulation will work with placeholder backgrounds. Replace these with actual PNG assets optimized for web (TinyPNG recommended).

Recommended dimensions: 640x500px for backgrounds, smaller for sprites.
